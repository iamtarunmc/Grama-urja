# **App Name**: Grama-Urja

## Core Features:

- Live Grid Monitor: A high-visibility dashboard providing real-time POWER ON/OFF status updates across villages using Firebase Realtime Database connections.
- Admin Control Center: Secure dashboard for administrators to manually toggle power status and onboard new villages to the grid.
- Village Directory & Selection: User-centric village selector that persists the user's home location locally for immediate access upon returning.
- Outage Predictor Tool: An AI-powered feature that uses historical power logs as a tool to forecast likely maintenance windows or potential grid strain alerts.
- Secure Auth Portal: Firebase-powered email/password authentication system to differentiate between village residents and grid administrators.
- Timestamped Activity Log: Automated ledger displaying precisely when power status was last modified to ensure data integrity.

## Style Guidelines:

- Primary Color: Deep Jade Green (#358a55) representing growth and grid health.
- Background Color: Soft Sage Tint (#f8faf9) to reduce glare in outdoor farming environments.
- Accent Color: Energetic Persimmon (#e36a44) for visibility on 'Power OFF' alerts and critical notifications.
- Font Recommendation: 'PT Sans', a humanist sans-serif chosen for its warmth and high readability in varying light conditions, making it ideal for rural users.
- Minimalist card-based layout optimized for mobile thumbs, with large high-contrast status indicators.
- Soft pulsing glow on the power indicator icons to indicate an active live-stream connection from the database.
- Heavy-stroke, clean-line icons for power icons and village markers to ensure clarity at all resolutions.
